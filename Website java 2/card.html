How to Play

  1. Each Player will have 250$ balance in their account in each Game.
  2. Start of each turn, 5$ will be deducted from each Player for Gamble.
  3. Each Player will be given 2 random cards at the start of each round.
  4. Player can specify the Gamble amount in their turn.
  5. After specifying Gamble amount, the player can Draw the card.
  6. If the Drawn card lies between of 2 given card, the players wins the specified amount of Gamble.
  7. If the Player fails to get the card in between of those 2 cards, the same amount of Gamble has to be paid.
  8. Each Player can Exchange one card with a random card by paying 5$ in each round.
  9. The Player having 0$ balance will be Eliminated from the Game.
  10. The last Player having none 0$ balance wins the Game.
